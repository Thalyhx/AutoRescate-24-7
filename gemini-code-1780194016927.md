# API de Referencia - Sistema AutoRescate 24/7

## 1. Estructuras de Datos Propias (Model)
*Estas clases son el núcleo de la restricción de CC1 (sin java.util).*

### ListaEnlazada<T>
* `agregar(T valor)`: Añade un elemento al final.
* `obtener(int indice)`: Retorna el elemento en la posición dada.
* `eliminar(T valor)`: Remueve un elemento y reajusta la lista.
* `contiene(T valor)`: Verifica existencia.
* `esVacia()`: Retorna true si no hay elementos.

### Cola<T>
* `encolar(T valor)`: Agrega al final.
* `desencolar()`: Retira del frente.
* `verFrente()`: Consulta sin retirar.

### Pila<T>
* `push(T valor)`: Inserta en el tope.
* `pop()`: Extrae del tope.
* `peek()`: Consulta el tope.

### ColaPrioridad
* `insertar(SolicitudServicio sol)`: Inserta manteniendo orden según prioridad.
* `extraer()`: Obtiene el de mayor prioridad.

## 2. Modelos de Negocio
*Clases que representan las entidades del sistema.*

### SolicitudServicio
* `getCliente()`, `getTipoEmergencia()`, `getEstado()`, `setEstado(EstadoSolicitud)`
* `getUbicacion()`, `setUnidadAsignada(UnidadServicio)`

### Kit
* `isCompleto()`, `setCompleto(boolean)`
* `getId()`, `getDescripcion()`

### PuestoAtencion
* `getSolicitudesPendientes()`: Retorna la ColaPrioridad.
* `buscarUnidad(TipoUnidad)`: Busca una unidad disponible en el puesto.
* `buscarTecnico(Especialidad)`: Busca un técnico disponible en el puesto.

---

## 3. Gestores (Controladores de Lógica)
*Clases Singleton que coordinan la operación.*

### GestorSolicitudes
* `crearSolicitud(Cliente c, TipoEmergencia t, String desc, PuntoVia ub, int zona)`
* `atenderSolicitud(SolicitudServicio sol)`: Realiza la asignación de recursos.
* `finalizarSolicitud(SolicitudServicio sol)`: Libera recursos y cierra caso.

### GestorKits
* `registrarKitEnRevision(Kit k)`: Mueve kit a zona de revisión.
* `liberarKitDeRevision()`: Mueve kit a estantería de listos.

### GestorOperaciones
* `registrarOperacion(String desc, String idAfec, String estAnt)`
* `removerUltimaOperacion()`: Implementa el deshacer (Pila).

### GestorPersistenciaJSON
* `guardarEstadoSimplificado()`: Exporta el estado actual a JSON.
* `leerEstadoGlobal()`: Lee el archivo JSON.

### ZonaCalculadora
* `calcularPuesto(PuntoVia punto)`: Devuelve el índice del puesto (0-3) más cercano.
